# HubX
Project Using DSA

HUBX Smart Warehouse is a console-based Java project developed using Data Structures and Object-Oriented Programming concepts.
The system helps manage warehouse products, perform searching/sorting operations, handle shipping orders, and maintain inventory efficiently.

This project demonstrates the practical implementation of:

1. Doubly Linked List
2. Stack
3. Queue
4. Searching
5. Sorting
6. Object-Oriented Programming (OOP)
